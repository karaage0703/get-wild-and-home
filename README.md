# get-wild-and-home
Get wild and tough!

# Setup

## Clone the repository
Execute following command:
```sh
$ git clone https://github.com/karaage0703/get-wild-and-home
```

## Get wild and home
Execute following commands:
```sh
$ cd get-wild-and-home
$ ./get_wild.sh
```
